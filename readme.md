# My novel reviews
A personal novel review showcase
## Tech stack
- HTML5
- CSS3
- JavaScript(ES6+)
- Font Awesome
- Particles.js
## Features
- Horizontal card carousel with mouse wheel scrolling
- Hamburger menu with smooth transition
- Genre filter buttons
- Hover effects on cards
- Animated particles background
- Responsive design
